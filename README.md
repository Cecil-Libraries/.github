# Cecil Libraries Organization
Welcome to the Cecil Libraries Organization's Home Page.

## Who are We?
We are a organization located in the USA, and founded by Chaosyr. Our focus is to create easy to use software for things like mods, games, applications and other forms of software.

## What is your Motto? What does it mean?
We aim to provide as much as we can (e.g. Helpers, Utilities, Examples, and so on), while keeping things simple within our libraries. You can find this in action within [Cecil.ANSI_Utils](https://github.com/Cecil-Libraries/Cecil-ANSI-Utils).
